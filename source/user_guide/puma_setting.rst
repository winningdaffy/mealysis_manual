PUMA设置
=================

PUMA设置窗口
--------------------------------

(a) 点击左侧工具条的 **PUMA** 按钮打开PUMA设置窗口，如 :numref:`img_puma_setting` 所示。

.. figure:: /images/puma_setting.png
    :width: 200px
    :align: center
    :name: img_puma_setting

    PUMA设置窗口

(c) 设置窗口中是PUMA数据列表，用于显示当前PUMA通信中的数据。
(d) PUMA数据列表中第一列为计算结果的名称，第二列为数据的单位，第三列为数据在PUMA系统中的名称
(e) 在数据列表区域点击右键，在弹出菜单中包括以下功能：

- 拷贝
- 粘贴
- 添加
- 删除
- 导入
- 清除所有
